Detailed Description

template<typename T, size_t size>
class CircularBuffer< T, size >

Circular buffer provides a ring buffer that is capable of writing and reading from opposite ends.

Please note that CircularBuffer is not considered thread-safe and must be synchronized externally if necessary.
